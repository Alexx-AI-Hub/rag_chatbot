# Local RAG Chatbot 🤖

This application is a local Retrieval-Augmented Generation (RAG) chatbot built mainly with Chainlit and LlamaIndex.

## How it works
- Answers are generated from your indexed local documents or using small Web Retrieval when switched on.
- You can upload temporary files during a session.
- Temporary files are automatically cleaned up on restart or chat end.
- Sources are shown when available.

## Usage
1. Ask a question about your indexed documents.
2. Upload files if you want to query additional content.
3. Review cited sources when provided.
